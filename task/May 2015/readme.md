### Continue技术小组五月份编程马拉松 – 项目篇

#### 基本要求

**考核范围：** 团队合作，GUI

**考核模式：** 项目成果展示

**组队人数：** 1 - 3 人

**考核时间：** 五月的第三个周日

**考核地点：** 不限（推荐软件学院113办公室）

**考核说明：** 不限制编程语言，不限制运行平台。最后考核以能运行起来的app为准。


#### 项目相关

**项目主题：** 小游戏 - 连连看

**基本功能：**
+ 时间限制
+ 游戏难度的设计

**考核标准：**
+ 游戏界面设计
+ 游戏可玩性
  - 难度设计
  - 关卡设计
+ 代码质量
